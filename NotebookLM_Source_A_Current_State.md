# DMAI Knowledge Engine: Current State Manifesto
## The Foundation of Architectural Intelligence

### 1. The Core Philosophy
The DMAI (DMAA Architectural Intelligence) Knowledge Engine is not simply a script or a basic configurator; it is the spatial backbone of the firm’s concept design phase. Developed to shape visionary environments, DMAI gives architects the immediate ability to configure programmatic massing, benchmark against global tech giants, and deploy vibrant, data-driven catalogs directly into the third dimension. 

### 2. Live Program Configuration
At its heart, DMAI handles complex programmatic logic with absolute ease. 
- **KPI Tracking:** Real-time calculation of Total Campus GFA, Target Employees, GFA/Employee, Floor Area Ratio (FAR), and Site Coverage.
- **Dynamic Program Allocation:** Users define typological shares (Offices, Landscape, Amenities, Housing, Mobility, Infrastructure) and watch as the system translates percentages into hard, required square meterage.
- **Aesthetic Data Visualization:** Built with curated DMAA color palettes, dynamic doughnut charts, and custom DINAMO typography, ensuring the data looks as premium as the architecture itself.

### 3. Integrated Global Benchmarking
Architects no longer guess if their spatial allocations are competitive. DMAI features a built-in benchmarking system.
- Instantly compare current project metrics against world-renowned campuses such as Apple Park, Microsoft Redmond, Googleplex, Meta Menlo Park, and the Nvidia Campus.
- A live comparison chart overlays the target GFA/Employee against these industry standards, bringing immediate grounding and authority to conceptual decisions.

### 4. Seamless 3D Translation & Export
Data does not remain trapped in tables.
- **Live 3D Preview:** The interface renders a real-time, interactive 3D module catalog. It transforms 2D programmatic spreadsheets into proportional physical volumes (extruding Length, Width, Height, and Floors) that users can orbit and inspect using preset top, front, and isometric camera angles.
- **Frictionless Export:** 
  - **OBJ Export:** One-click download of the 3D module catalog, instantly ready to drag and drop into Rhino, SketchUp, or Blender.
  - **PPTX Generation:** Automated creation of a branded, 3-slide PowerPoint pitch deck directly from the interface.
  - **CSV Export:** Outputs exact area schedules for Revit, QS workflows, and Excel.

### 5. The Worldwide Portfolio & Assistant
- **World Map:** A global dashboard mapping all historic and ongoing DMAA projects side-by-side, tracking the firm's footprint.
- **AI Assistant:** An integrated LLM-powered bot that allows architects to "talk to their projects"—querying project history, typologies, and firm knowledge securely within the browser.
